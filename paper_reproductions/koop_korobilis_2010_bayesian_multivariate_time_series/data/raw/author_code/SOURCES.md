# Author Code Sources

Downloaded on 2026-05-19 from Dimitris Korobilis's Matlab code page:

https://sites.google.com/site/dimitriskorobilis/matlab/code-for-vars

The page states that the BVAR programs are the programs for empirical
illustrations in Koop and Korobilis (2010), "Bayesian Multivariate Time Series
Methods for Empirical Macroeconomics".

Downloaded packages include:

- `BVAR_Analytical.zip`
- `BVAR_FULL.zip`
- `BVAR_GIBBS.zip`
- `BVAR_SSVS.zip`
- `BVAR_VARIABLE_SELECTION.zip`
- `TVP_VAR_CK.zip`
- `TVP_VAR_DK.zip`
- `TVP_VAR_MIX.zip`
- `HIERARCHICAL_TVP_VAR.zip`
- `FACTOR_MODELS.zip`
- `FAVAR.zip`
- `TVP_FAVAR.zip`

The local R replication uses `BVAR_FULL/Yraw.dat` and `BVAR_FULL/yearlab.dat`
as the table-matching author data.
